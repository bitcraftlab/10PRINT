
10 PRINT
========

[![10 Print Book Cover](http://www.creativeapplications.net/wp-content/uploads/2012/12/10PRINT_01.jpg)][Link01]

### Code

    10 PRINT CHR$(205.5+RND(1)) GOTO 10
